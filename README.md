## ✨ Features

- **3D Animated Hero** — Rotating icosahedron rendered in real-time with Canvas 2D API, reacting to mouse movement
- **Particle Starfield Background** — Perspective 3D particle field with mouse parallax interaction
- **Animated Project Cards** — Each card has a unique live canvas animation (wave, nebula, grid dots, vortex)
- **Scroll-Triggered Skill Bars** — Animated progress bars fire on scroll using IntersectionObserver
- **Count-Up Stats** — Numbers animate up when scrolled into view
- **Custom Cursor** — Magnetic cursor with blend-mode difference effect
- **Glitch Text Effect** — Hero name uses CSS glitch animation with pseudo-elements
- **Responsive Design** — Fully mobile-friendly with a hamburger nav and fluid `clamp()` typography
- **Scroll Progress Bar** — Thin gradient bar at the top tracks reading progress
- **Loading Screen** — Animated loader with percentage counter before the portfolio reveals
- **Zero Dependencies** — No frameworks, no libraries — pure vanilla HTML/CSS/JS, single file

## 🚀 Getting Started

### View Locally

```bash
# Clone the repository
git clone https://github.com/wasiakbar8/wasi-portfolio.git

# Navigate into the folder
cd wasi-portfolio

# Open in browser (or use Live Server in VS Code)
open index.html
```

> ⚠️ For the profile image and CV download to work, make sure `profile.jpeg` and `my-cv.pdf` are in the same folder as `index.html`.

### Deploy to Vercel

```bash
# Install Vercel CLI
npm i -g vercel

# Deploy
vercel
```

Or simply drag and drop the folder into [vercel.com/new](https://vercel.com/new).
